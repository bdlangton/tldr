# vim location window

> Location window functionality in vim.
> [ and ] are vim-unimpaired bindings.

- Toggle the location window (custom mapping)

`<leader>cl`

- Open location window

`:lope`

- Close location window

`:lcl`

- Go to previous location

`:lp OR [l`

- Go to next location

`:lne OR ]l`

- Go to first location

`:lfir OR [L`

- Go to last location

`:lla OR ]L`
