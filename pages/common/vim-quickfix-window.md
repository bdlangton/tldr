# vim quickfix window

> Quickfix window functionality in vim.
> [ and ] are vim-unimpaired bindings.

- Toggle the quickfix window (custom mapping)

`<leader>cq`

- Open quickfix window

`:cope`

- Close quickfix window

`:ccl`

- Go to previous location

`:cp OR [q`

- Go to next location

`:cn OR ]q`

- Go to first location

`:cfir OR [Q`

- Go to last location

`:cla OR ]Q`
